양의 정수 판별

방법

📖
정수는 1로 나누었을 경우 항상 나머지가 0인점을 판별

코드

function isInteger(number:number) :Boolean
{
	return number%1 ? false : true
}

isInteger(12) // true
isInteger(123.1) // false
isInteger(-10) // false
